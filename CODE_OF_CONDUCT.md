# Code of Conduct

## Our Commitment

We are committed to making participation in the Apex project a harassment-free experience for everyone, regardless of:

- Age
- Gender
- Identity
- Experience level
- Nationality
- Background

We value constructive collaboration.

---

## Expected Behavior

Participants are expected to:

- Be respectful
- Provide constructive feedback
- Focus on technical merit
- Encourage new contributors
- Accept differing viewpoints professionally

---

## Unacceptable Behavior

The following will not be tolerated:

- Harassment or intimidation
- Personal attacks
- Discriminatory language
- Public shaming
- Deliberate disruption of discussions

---

## Enforcement

Project maintainers reserve the right to:

- Remove comments
- Reject contributions
- Temporarily or permanently ban participants

Enforcement decisions will prioritize project health and contributor safety.

---

## Reporting

If you experience or witness unacceptable behavior, open a private issue or contact the maintainers directly.

All reports will be handled confidentially.

---

## Scope

This Code of Conduct applies to:

- GitHub discussions
- Issues and PRs
- Project communication channels

---

Apex is built on professionalism and respect.
